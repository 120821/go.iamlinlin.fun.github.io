<p>2.下载后进行解压缩</p>

<p>unzip xxx</p>

<p>然后安装</p>

<pre>
<code>sudo dpkg -i fcitx-baidupinyin.deb{% endhighlight %}

<p>3，如果安装出现错误尝试：</p>

<pre>
<code>sudo apt update{% endhighlight %}

<p>提示：</p>

<pre>
<code>sudo dpkg -i fcitx-baidupinyin.deb
