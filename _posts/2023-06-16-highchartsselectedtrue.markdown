{% highlight %}
{% highlight %}data: [{ name: &#39;面粉:223吨&#39;, y: 4.6 }]{% endhighlight %}

<p>请注意， {% highlight %}name{% endhighlight %} 和 {% highlight %}y{% endhighlight %} 每个数据点仍然需要属性。</p>
