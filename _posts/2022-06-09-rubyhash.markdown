                    <p>实现这样的方法： 可以动态创建一个二维Hash, 内容如下</p> 
<pre><code>generate_hash(3) 

获得：

{
  "parent1": {
      "child1": "value1"
  },
  "parent2": {
      "child2": "value2"
  },
  "parent3": {
      "child3": "value3"
  }
}

再如：

generate_hash(5) 

获得：

{
  "parent1": {
      "child1": "value1"
  },
  "parent2": {
      "child2": "value2"
  },
  "parent3": {
      "child3": "value3"
  },
  "parent4": {
      "child4": "value4"
  },
  "parent5": {
      "child5": "value5"
  }
}</code></pre> 
<pre><code class="language-ruby">def generate_hash(n)
  result = {}
  for i in (1..n)
    hash = {"child#{i}" =&gt; "value#{i}"}
    for i in (1..i)
      result["parent#{i}"] = hash
    end 
  end 
  return result
end

puts generate_hash(5)

</code></pre>
