<p>实现上下居中：</p>

<p>在css标签内，再使用position设置div位置属性为absolute绝对定位，使用top设置距离上边缘为50%，使用left设置距离左边缘为50%，因为要实现居中，因此必须减掉div自身宽度和高的一半（margin:-100px 0 0 -150px），从而实现上下左右居中。</p>

<p>所以也可以：</p>
